5 html tags learned
head tag
head tag is used to define the heading of the html code.
title tag
title tag is used to define the title of the html code, which is shown in the title's bar.
paragraph tag
paragraph tag is used to define the paragraph of the html.
line breaker tag
line breaker tag is used to break a line in the html code.
body tag
the body tag is where you insert the body content of the web page's content.
